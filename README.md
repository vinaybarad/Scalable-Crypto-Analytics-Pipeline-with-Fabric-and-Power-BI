**Scalable Crypto Analytics Pipeline with Microsoft Fabric and Power BI**


Designed and developed an end-to-end Scalable Crypto Analytics Pipeline using Microsoft Fabric, Power BI, and Power Automate to automate cryptocurrency data ingestion, transformation, storage, monitoring, and visualization. The project focused on building a modern cloud-based analytics architecture capable of handling automated data refreshes, centralized storage, real-time reporting, and failure notifications.


**Project Overview :**

The project involved extracting real-time cryptocurrency market data from external APIs such as the CoinGecko API
 and building a complete data analytics workflow using Microsoft Fabric. The pipeline automates data ingestion, stores processed data in a centralized Data Warehouse, creates semantic models for business reporting, and visualizes insights through interactive Power BI dashboards published to Power BI Service.

**End-to-End Workflow:**

->Extracted cryptocurrency market data from external REST APIs including CoinGecko API

->Built automated data pipelines in Microsoft Fabric for scheduled data ingestion and processing

->Performed data transformation, cleansing, and validation within Fabric pipelines

->Designed and implemented a centralized Data Warehouse in Microsoft Fabric to store historical and real-time crypto data

->Created semantic models to optimize analytical querying and reporting performance

->Connected Power BI Desktop to Fabric Semantic Models for report development

->Developed interactive Power BI dashboards to analyze:

->Cryptocurrency price trends

->Market capitalization

->Trading volume

->Top-performing coins

->Historical performance analysis

->KPI monitoring and trend visualization

->Published dashboards and reports to Power BI Service for cloud-based access and sharing

->Configured scheduled refresh automation:
Pipeline refresh scheduled daily at 8:00 AM
Power BI dashboard refresh scheduled automatically at 8:30 AM

->Implemented monitoring and alert mechanisms using Power Automate

->Configured automated email notifications to alert users whenever pipeline refresh failures or workflow issues occur

->Ensured reliable and scalable analytics processing using Microsoft Fabric cloud architecture
